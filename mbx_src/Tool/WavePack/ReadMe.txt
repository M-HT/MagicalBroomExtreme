=====================================================================
	説明
=====================================================================
　DirectX8用簡易ライブラリLunaで使用可能なWaveファイルのパック化ツールです。
　大量のWaveを置いておくのが嫌な場合や、
　Waveファイルを見られたくない場合に有効です。
　余計なチャンクは一切省いてありますので、多少はサイズが小さくなるようです。

=====================================================================
	使用方法
=====================================================================
　パック化したいファイルを入れてあるディレクトリで起動してください。
　自動的にフォルダ内のファイルをパック化します。
　サブフォルダは検索しません。

=====================================================================
	インストール・アンインストール
=====================================================================
　レジストリは一切いじってないので、自由にコピー・削除してもらってOKです。
